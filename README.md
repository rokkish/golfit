### how to setup

```sh
# 0. clone the code
git clone git@github.com:rokkish/golfit.git

# 1. Add golfit function to your shell file
cat golfit.sh >> ~/.zshrc

# 2. Rerun your terminal
source ~/.zshrc

# 3. And You can play!
golfit

```

### how to play
```sh
# 1. input s and Enter
input s and start the roulette >>> 

# 2. Press Ctrl+C for stop

```
